六芒星雷达 — HexagramRadar.tsx

右下角固定浮层 · SVG 六边形雷达图
        🏆 总冠军    ──── 3
        👑 历史地位   ──── 78
        📺 媒体好感   ──── 92  ← 加盟纽约后飙升
        ❤️ 球迷口碑   ──── 65
        💰 薪资健康   ──── 41  ← 大合同吃空间
        ⚡ 体能透支   ──── 35  ← 还好，没到后期

SVG 实现方案：

  1. 6 条轴线从中心发出，角度间隔 60°
  2. 每条轴标注：图标 + 名称 + 最大值
  3. 当前数值连成的六边形（fill: 半透明团队色）
  4. 数值变动时：GSAP 动画顶点位置平滑过渡（duration: 0.6s, ease: elasticOut）
  5. 颜色编码：每个顶点独立着色 — 绿色(>70) / 黄色(40-70) / 红色(<40)
  6. 数值旁边显示 delta 标签：+25 绿色闪烁、-15 红色闪烁，1.5s 后消失

交互行为：

  • LLM 生成 3-5 条后，以 200ms stagger 间隔从顶部滑入
  • anim: translateX(-100% → 0), opacity 0→1, duration 0.4s
  • 新帖子到达时旧帖向下推（自然瀑布流）
  • 悬停时卡片微微放大 + 停止自动滚动
  • 情感色编码边框：angry=红 / excited=金 / sarcastic=紫 / shocked=蓝
  • 5s 后旧帖自动淡出（除非用户悬停），最多同时显示 5 条
  • 底部小字："以上内容由 AI 生成，仅为娱乐效果"
