本项目旨在构建一个高度交互式的“超级巨星职业生涯模拟器”。有别于传统的对话式机器人或静态时间轴图表，本项目引入了 RPG（角色扮演）游戏机制与 3D 可视化拓扑图，允许用户在历史的关键分叉点（如 2010 年“决定一”）替勒布朗·詹姆斯做出选择，并通过大模型（LLM）实时演算出全新的平行宇宙时间线。

项目目标：
构建一个视觉表现力极强、情绪互动拉满的 Web3D/AI 结合应用。底层架构需体现对复杂数据流转和状态机管控的深度理解，作为极具竞争力的工程化落地案例展示。

2. 核心玩法与前端表现层 (UI/UX)
2.1 3D 命运星图 (The 3D Destiny Graph)
彻底摒弃 2D 连线，采用全屏 3D 力导向图展示历史脉络。

视觉规范： 基于 three.js 和 3d-force-graph。背景为深邃的星空。

节点定义 (Nodes)： 核心节点是詹姆斯，周围环绕着不同的星团（热火时代、骑士时代、宿敌如库里/杜兰特）。

连线定义 (Links)： 节点间的连线带有粒子流动光效，粗细代表羁绊深度。

交互动画： 当用户修改了 2010 年的决定（例如去了尼克斯），代表真实历史的热火星团瞬间失去引力，化为粒子消散；AI 实时推演的新节点（尼克斯宇宙）在暗处迅速聚合发光，并重新建立引力连线。

2.2 动态六芒星雷达 (RPG 状态引擎)
在界面右下角固定一个实时变动的个人属性面板，用户的每一次抉择都会引发数值的剧烈变动。

追踪维度： 总冠军数、历史地位 (Legacy)、媒体好感度、球迷口碑、球队薪资空间、体能透支率。

机制： 数值面板的更新必须与星图的重组同步发生，打破“与 AI 聊天”的错觉，建立“掌控复杂系统”的游戏感。

2.3 虚拟舆论瀑布流 (Social Media Backlash)
在界面左侧边栏引入虚拟社交媒体反应。

机制： 当生成一条具有争议性的平行时间线（如联手公牛巅峰罗斯）时，调用大模型同步生成 3-5 条模拟真实社交网络情绪的“虚拟推特”（包含名嘴狂喷、极端球迷恶搞等），以瀑布流形式刷出。

3. 后端架构与 AI 引擎设计 (Backend & AI)
本系统采用单次全量生成 + 流式输出 (SSE) 的轻量化 MVP 架构，坚决不使用 RAG 以防止相似度召回导致的历史串线幻觉。

3.1 核心技术栈
前端： React + TypeScript + 3d-force-graph (WebGL) + Tailwind CSS

后端 API 服务： Python (FastAPI) 或轻量级 Node.js。需提供强并发控制，并实现基于 Server-Sent Events (SSE) 的流式接口。

AI 引擎： Claude 3.5 Sonnet (或其他具备极强长文本逻辑推演能力的顶级模型)。

3.2 防幻觉机制 (Anti-Hallucination Protocol)
全量依赖“硬编码上下文 + 严格 Schema”来驯服大模型：

铁律注入： 在 System Prompt 中硬编码 2010 年的真实历史状态（薪资帽、各队真实大名单、自由市场可用球员）。

强类型输出约束： 要求大模型必须返回符合 Pydantic/Zod 预定义的 JSON 结构数组。绝对禁止包含 Markdown 代码块标记之外的废话。

后端实体拦截： 在 SSE 推送给前端前，后端利用预设的白名单进行校验（如果 AI 生成了不存在的球员，立刻抛弃并触发重试机制）。

4. MVP 实施范围 (The Tutorial Level)
为了在 7-10 天内跑通全链路，MVP 版本仅包含一个剧情分叉点。

锚点事件： 2010 年 7 月 8 日，The Decision。巨星的命运是相互纠缠的。如果 2016 年 KD 没有加入勇士，詹姆斯的 2017 和 2018 年大概率会多拿冠军，欧文可能就不会觉得“夺冠无望/想当老大”而申请交易，骑士王朝可能还会延续。 这种“牵一发而动全身”的宿命碰撞，正是这个项目的终极魅力。

要实现这个，你的底层 AI Prompt 逻辑需要加一个小小的“外挂”：全局世界状态 (Global World State)。当你让 AI 推演詹姆斯的剧本时，必须把当前宇宙里 KD 的状态作为前置条件塞进去（反之亦然）。大模型极度擅长处理这种“蝴蝶效应”的逻辑推演。

关于詹姆斯职业生涯中那些足以“撕裂时间线”的关键分叉点，除了 2010 年的“决定一”，以下这几个节点绝对是必加入的核弹级大戏：

👑 勒布朗·詹姆斯多重宇宙：核心命运分叉点
1. 2014年：“决定二” (The Return vs. The Dynasty)
真实历史： 离开热火，发表《我回家了》公开信，重返克利夫兰骑士。

平行选项 A（留守迈阿密）： 顶薪续约热火。韦德的膝盖继续恶化，帕特·莱利强行交易波什换取新鲜血液。热火在抗击勇士崛起的过程中走向何方？

平行选项 B（航天城的三巨头）： 加盟休斯顿火箭。当时火箭拥有詹姆斯·哈登和德怀特·霍华德，且疯狂追求过詹姆斯。如果他去了西部，NBA 历史将彻底改写。

2. 2014年：维金斯与乐福的交易博弈 (The Youth Movement)
真实历史： 逼迫骑士管理层用状元安德鲁·维金斯换取森林狼的凯文·乐福，组成骑士三巨头。

平行选项 A（保留维金斯）： 詹姆斯决定亲自调教这位“枫叶乔丹”和欧文。骑士依靠年轻的运动天赋冲击冠军。

平行选项 B（截胡保罗·乔治）： 骑士用状元签和选秀权去向步行者报价保罗·乔治（当时有此流言）。

3. 2017年：欧文的决裂 (The Kyrie Schism)
真实历史： 欧文申请交易，骑士将其送至波士顿凯尔特人，换来小托马斯和篮网签（塞克斯顿）。这直接导致了骑士 2018 年的艰难与詹姆斯的最终离开。

平行选项 A（强压怒火，拒绝交易）： 詹姆斯和管理层强行留下欧文，并在更衣室召开闭门会议。2018 年的“全力詹”如果身边有欧文，能复仇 KD 的勇士吗？

平行选项 B（保罗·乔治/血布大交易）： 当时险些成行的三方交易：欧文去太阳，保罗·乔治和布莱德索来骑士。詹姆斯搭档巅峰 PG13。

4. 2018年：“决定三” (The Hollywood Move vs. The Process)
真实历史： 加盟洛杉矶湖人，开启好莱坞时代。

平行选项 A（相信过程）： 加盟费城 76 人。当时本·西蒙斯和恩比德正值青春风暴，詹姆斯的加入将组成东部最恐怖的前场。

平行选项 B（休斯顿的魔球巅峰）： 加盟火箭，联手保罗和哈登。2018 年这支差点掀翻宇宙勇的火箭，如果加上勒布朗，能否建立王朝？

5. 2021年：威少交易之谜 (The Westbrook Disaster)
真实历史： 放弃希尔德，推动湖人梭哈交易来拉塞尔·威斯布鲁克。这被认为是詹姆斯生涯最糟糕的幕后操作之一，直接摧毁了湖人的争冠窗口。

平行选项 A（希尔德与特纳）： 湖人按原计划完成与国王/步行者的交易，获得极致空间（希尔德）和护框（特纳）。詹姆斯和浓眉在完美的阵容搭配下能否拿下第五冠？

平行选项 B（签下德罗赞）： 德罗赞当时极度渴望加盟湖人。如果是德罗赞而不是威少到来，湖人的侧翼火力会如何进化？

把杜兰特的线也加进来之后，这个项目的星图在视觉上会极其宏大。你可以想象画面里有两颗最亮的超大恒星（詹姆斯和杜兰特），它们各自带着自己的星团，并且在 2012年总决赛、2017-2018年骑勇大战时，这两颗恒星的连线会爆发出剧烈的光芒。
